Курс HTML

Текстовые элементы

Тег <html> этот тег говорит нам о том, сейчас будет писаться код.это корневой элемент HTML-документа. Все остальные элементы должны содержаться внутри этого корневого элемента.

Пример: <html></html> см. ссылку вижу

Посмотреть пример наглядно

Тег <p> используется для определения абзаца текста.

Пример: <p>Привет, мир!</p>

Посмотреть пример наглядно

Тег <h1> определяет заголовок раздела первого уровня : наивысший уровень.

Пример: <h1>Мой блог</h1>

Посмотреть пример наглядно

Тег <span>определяет общий встроенный контейнер контента, не несущий никакой семантической ценности.

Пример: <span>Мой блог</span>

Тег <pre>Определяет предварительно отформатированный текст.

Пример: <pre>

                Идёт направо — 
                          песнь заводит,
                Налево — 
                         сказку говорит.
                Там чудеса: там леший бродит,
                Русалка на ветвях сидит;
</pre>
