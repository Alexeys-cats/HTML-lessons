Основные тэги/структура страницы

Основные теги HTML, которые используются для создания структуры веб-страницы.

Тег <!DOCTYPE>

Описание: Указывает тип документа и версию HTML, которую использует страница.

Пример: <!DOCTYPE html>

Посмотреть пример

Тег <html>

Описание: Корневой элемент HTML-документа, внутри которого находятся все остальные элементы страницы.

Пример: <html></html>

Посмотреть пример

Тег <head>

Описание: Содержит мета-информацию о документе, такую как заголовок, ссылки на стили, скрипты и другие мета-данные.

Пример: <head></head>

Посмотреть пример

Тег <body>

Описание: Основное содержимое веб-страницы, которое отображается на экране.

Пример: <body>Контент страницы</body>

Посмотреть пример

Тег <title>

Описание: Определяет название документа, которое отображается в заголовке окна браузера или вкладке.

Пример: <title>Моя страница</title>

Посмотреть пример

Тег <meta>

Описание: Используется для задания мета-данных, таких как кодировка, описание страницы, ключевые слова и другие данные.

Пример: <meta charset="UTF-8">

Посмотреть пример

Тег <link>

Описание: Используется для связи документа с внешними ресурсами, такими как стили CSS.

Пример: <link rel="stylesheet" href="styles.css">

Посмотреть пример

Тег <script>

Описание: Используется для вставки и выполнения JavaScript-кода на странице.

Пример: <script>console.log("Привет, мир!");</script>

Посмотреть пример

Тег <a>

Описание: Определяет гиперссылку, которая может вести на другую страницу или ресурс.

Пример: <a href="https://example.com">Перейти на сайт</a>

Посмотреть пример

Тег <p>

Описание: Определяет абзац текста.

Пример: <p>Привет, мир!</p>

Посмотреть пример

Тег <h1>

Описание: Определяет заголовок первого уровня — наивысший уровень заголовка на странице.

Пример: <h1>Мой блог</h1>

Посмотреть пример

Тег <h2>

Описание: Определяет заголовок второго уровня, ниже заголовка первого уровня.

Пример: <h2>Моя статья</h2>

Посмотреть пример

Тег <h3>

Описание: Определяет заголовок третьего уровня, используется для подразделов.

Пример: <h3>Введение</h3>

Посмотреть пример

Тег <h4>

Описание: Определяет заголовок четвёртого уровня, используется для более мелких подразделов.

Пример: <h4>Тема 1</h4>

Посмотреть пример

Тег <h5>

Описание: Определяет заголовок пятого уровня, обычно используется для ещё более мелких подразделов.

Пример: <h5>Подтема 1.1</h5>

Посмотреть пример

Тег <h6>

Описание: Определяет заголовок шестого уровня, обычно используется для наименее важных подразделов.

Пример: <h6>Подтема 1.1.1</h6>

Посмотреть пример

Тег <ul>

Описание: Определяет ненумерованный список.

Пример: <ul><li>Элемент списка</li></ul>

Посмотреть пример

Тег <ol>

Описание: Определяет нумерованный список.

Пример: <ol><li>Первый элемент</li></ol>

Посмотреть пример

Тег <li>

Описание: Определяет элемент списка, используется внутри <ul> или <ol>.

Пример: <li>Элемент списка</li>

Посмотреть пример

Тег <img>

Описание: Используется для вставки изображения на страницу.

Пример: <img src="image.jpg" alt="Описание изображения">

Посмотреть пример

Тег <table>

Описание: Используется для создания таблицы.

Пример: <table><tr><td>Ячейка 1</td><td>Ячейка 2</td></tr></table>

Посмотреть пример

Тег <tr>

Описание: Определяет строку таблицы, используется внутри тега <table>.

Пример: <tr><td>Ячейка 1</td><td>Ячейка 2</td></tr>

Посмотреть пример

Тег <td>

Описание: Определяет ячейку таблицы, используется внутри тега <tr>.

Пример: <td>Содержимое ячейки</td>

Посмотреть пример

Тег <th>

Описание: Определяет заголовочную ячейку таблицы. Обычно используется для заголовков столбцов или строк.

Пример: <th>Заголовок</th>

Посмотреть пример

Тег <caption>

Описание: Добавляет заголовок к таблице. Он будет располагаться над таблицей.

Пример: <caption>Моя таблица</caption>

Посмотреть пример

Тег <form>

Описание: Определяет форму для ввода данных от пользователя, которая может отправлять эти данные на сервер.

Пример: <form action="submit_form.php"><input type="text"></form>

Посмотреть пример

Тег <input>

Описание: Определяет поле для ввода данных. Используется внутри формы для ввода текста, чисел и других типов данных.

Пример: <input type="text" name="username">

Посмотреть пример

Тег <button>

Описание: Определяет кнопку, которую можно использовать в формах или для других действий на странице.

Пример: <button type="submit">Отправить</button>

Посмотреть пример

Тег <select>

Описание: Создаёт выпадающий список для выбора значений.

Пример: <select><option>Первый вариант</option><option>Второй вариант</option></select>

Посмотреть пример

Тег <option>

Описание: Определяет элемент в выпадающем списке, созданном с помощью тега <select>.

Пример: <option value="1">Первый вариант</option>

Посмотреть пример

Тег <div>

Описание: Используется для группировки элементов в контейнер. Очень часто применяется для стилизации или структурирования страницы.

Пример: <div class="container">Контент</div>

Посмотреть пример

Тег <span>

Описание: Лёгкий, строчный контейнер для текста или других элементов. Обычно используется для применения стилей к части текста.

Пример: <span class="highlight">Текст</span>

Посмотреть пример

Тег <strong>

Описание: Определяет текст, который нужно выделить жирным шрифтом. Обычно используется для выделения важного контента.

Пример: <strong>Важное сообщение</strong>

Посмотреть пример

Тег <em>

Описание: Определяет текст, который следует выделить курсивом. Обычно используется для выделения важности текста.

Пример: <em>Это важно</em>

Посмотреть пример

Тег <br>

Описание: Вставляет разрыв строки в тексте, не создавая нового абзаца.

Пример: Текст перед разрывом<br>Текст после разрыва

Посмотреть пример

Тег <hr>

Описание: Вставляет горизонтальную линию (разделитель), обычно используется для отделения блоков контента.

Пример: <hr>

Посмотреть пример

Тег <blockquote>

Описание: Используется для выделения цитаты или длинного фрагмента текста. Обычно текст цитаты размещается с отступом.

Пример: <blockquote>Это цитата</blockquote>

Посмотреть пример

Тег <q>

Описание: Определяет короткую цитату. В отличие от <blockquote>, не делает отступа для текста.

Пример: <q>Это короткая цитата</q>

Посмотреть пример

Тег <code>

Описание: Определяет фрагмент кода. Обычно используется для отображения кода программирования.

Пример: <code>console.log("Hello, World!");</code>

Посмотреть пример

Тег <pre>

Описание: Определяет текст, который должен быть отображён в фиксированном формате, включая пробелы и разрывы строк.

Пример: <pre> Это пример текста с пробелами</pre>

Посмотреть пример

Тег <a>

Описание: Создаёт гиперссылку на другую страницу, сайт или якорь внутри документа.

Пример: <a href="https://example.com">Перейти на сайт</a>

Посмотреть пример

Тег <img>

Описание: Отображает изображение на странице. Для указания пути к файлу используется атрибут src.

Пример: <img src="image.jpg" alt="Описание изображения">

Посмотреть пример

Тег <figure>

Описание: Используется для группировки медиа-объектов с подписями — изображений, графиков, иллюстраций.

Пример: <figure><img src="image.jpg"><figcaption>Описание</figcaption></figure>

Посмотреть пример

Тег <figcaption>

Описание: Добавляет подпись для содержимого внутри <figure>.

Пример: <figcaption>Это подпись изображения</figcaption>

Посмотреть пример

Тег <nav>

Описание: Определяет блок для навигации по сайту, в котором обычно размещают меню ссылок.

Пример: <nav><a href="/">Главная</a></nav>

Посмотреть пример

Тег <header>

Описание: Определяет "шапку" документа или раздела. Чаще всего содержит логотип, название и меню навигации.

Пример: <header><h1>Заголовок сайта</h1></header>

Посмотреть пример

Тег <footer>

Описание: Определяет "подвал" документа или секции. Обычно содержит контактную информацию и копирайты.

Пример: <footer>© 2025 Все права защищены</footer>

Посмотреть пример

Тег <section>

Описание: Разделяет контент на логические блоки, которые обычно объединены общей темой.

Пример: <section><h2>Раздел 1</h2></section>

Посмотреть пример

Тег <article>

Описание: Определяет самостоятельный и законченый контент, например новость, пост в блоге или комментарий.

Пример: <article><h2>Заголовок статьи</h2></article>

Посмотреть пример

Тег <aside>

Описание: Определяет контент, косвенно связанный с основным содержимым. Обычно используется для боковых панелей с дополнительной информацией.

Пример: <aside>Полезная информация</aside>

Тег <tr>

Описание: Определяет строку таблицы.

Пример: <tr><td>Текст</td></tr>

Посмотреть пример

Тег <td>

Описание: Создаёт ячейку с данными внутри строки таблицы.

Пример: <td>Данные</td>

Тег <th>

Описание: Определяет заголовочную ячейку таблицы (обычно выделяется жирным и выравнивается по центру).

Пример: <th>Заголовок</th>

Тег <thead>

Описание: Группирует заголовочную часть таблицы.

Пример: <thead><tr><th>Название</th></tr></thead>

Посмотреть пример

Тег <tbody>

Описание: Группирует основное содержимое таблицы.

Пример: <tbody><tr><td>Данные</td></tr></tbody>

Посмотреть пример

Тег <tfoot>

Описание: Группирует нижнюю часть таблицы, обычно используется для итоговых строк.

Пример: <tfoot><tr><td>Итого</td></tr></tfoot>

Посмотреть пример

Тег <form>

Описание: Контейнер для формы, в котором размещаются поля для ввода, кнопки и другие элементы управления.

Пример: <form action="/submit"><input type="text"></form>

Посмотреть пример

Тег <input>

Описание: Поле для ввода данных. Может быть текстовым, числовым, чекбоксом, радиокнопкой и др.

Пример: <input type="text" placeholder="Ваше имя">

Посмотреть пример

Тег <label>

Описание: Подпись для поля формы, связывается с элементом с помощью атрибута for.

Пример: <label for="name">Имя:</label><input id="name" type="text">
