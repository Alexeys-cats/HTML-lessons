Классы

Основные свойства CSS, которые часто используются в веб-разработке.

Свойство color

Описание: Определяет цвет текста элемента.

Пример: p { color: red; }

Свойство background-color

Описание: Устанавливает цвет фона для элемента.

Пример: div { background-color: #f0f0f0; }

Документация

Свойство font-size

Описание: Определяет размер шрифта текста.

Пример: h1 { font-size: 32px; }

Документация

Свойство font-weight

Описание: Задаёт толщину шрифта (жирность).

Пример: strong { font-weight: bold; }

Документация

Свойство text-align

Описание: Определяет выравнивание текста внутри элемента.

Пример: p { text-align: center; }

Документация

Свойство margin

Описание: Устанавливает внешний отступ элемента (расстояние до других элементов).

Пример: div { margin: 20px; }

Документация

Свойство padding

Описание: Определяет внутренний отступ между содержимым и границей элемента.

Пример: div { padding: 10px; }

Документация

Свойство border

Описание: Добавляет рамку вокруг элемента.

Пример: div { border: 1px solid black; }

Документация

Свойство display

Описание: Определяет тип отображения элемента (блочный, строчный, flex, grid и т.д.).

Пример: div { display: flex; }

Документация

Свойство position

Описание: Определяет способ позиционирования элемента на странице (static, relative, absolute, fixed, sticky).

Пример: div { position: absolute; top: 10px; left: 20px; }

Документация

Свойства top, right, bottom, left

Описание: Задают смещение элемента при относительном, абсолютном или фиксированном позиционировании.

Пример: div { position: absolute; top: 50px; left: 100px; }

Документация

Свойство flex

Описание: Задает, как элемент будет сжиматься, расширяться и занимать пространство в flex-контейнере.

Пример: div { flex: 1 1 auto; }

Документация

Свойство justify-content

Описание: Определяет горизонтальное выравнивание элементов внутри flex- или grid-контейнера.

Пример: .container { display: flex; justify-content: center; }

Документация

Свойство align-items

Описание: Определяет вертикальное выравнивание элементов внутри flex-контейнера.

Пример: .container { display: flex; align-items: center; }

Документация

Свойство z-index

Описание: Определяет порядок наложения элементов (чем больше значение — тем выше элемент).

Пример: div { position: relative; z-index: 10; }

Документация

Свойство overflow

Описание: Управляет отображением содержимого, выходящего за пределы элемента (visible, hidden, scroll, auto).

Пример: div { overflow: hidden; }

Документация

Свойство object-fit

Описание: Определяет, как содержимое (обычно изображения или видео) вписывается в контейнер.

Пример: img { object-fit: cover; }

Документация

Свойство opacity

Описание: Управляет прозрачностью элемента. Значения от 0 (полностью прозрачный) до 1 (полностью видимый).

Пример: div { opacity: 0.5; }

Документация

Свойство transition

Описание: Задает параметры анимации при изменении свойств элемента.

Пример: div { transition: all 0.3s ease; }

Документация

Свойство transform

Описание: Позволяет изменять форму и положение элемента: поворачивать, сдвигать, масштабировать и т.д.

Пример: div { transform: rotate(45deg); }

Документация

Свойство box-shadow

Описание: Добавляет тень к элементу.

Пример: div { box-shadow: 0 4px 8px rgba(0,0,0,0.2); }

Документация

Свойство border-radius

Описание: Делает углы элемента скругленными.

Пример: div { border-radius: 8px; }

Документация

Свойство background-image

Описание: Устанавливает изображение в качестве фона для элемента.

Пример: div { background-image: url('image.jpg'); }

Документация

Свойство cursor

Описание: Определяет вид курсора при наведении на элемент.

Пример: a { cursor: pointer; }

Документация

Свойство visibility

Описание: Управляет видимостью элемента, не удаляя его из документа. Значения: visible, hidden, collapse.

Пример: div { visibility: hidden; }

Документация

Свойство pointer-events

Описание: Управляет обработкой событий мыши для элемента.

Пример: div { pointer-events: none; }

Документация

Свойство white-space

Описание: Управляет тем, как браузер обрабатывает пробелы внутри элемента.

Пример: p { white-space: nowrap; }

Документация

Свойство word-wrap

Описание: Управляет переносом длинных слов внутри контейнера.

Пример: div { word-wrap: break-word; }

Документация

Свойство clip-path

Описание: Ограничивает область видимости элемента по заданной фигуре.

Пример: img { clip-path: circle(50%); }

Документация

Свойство filter

Описание: Применяет графические эффекты (размытие, контраст и т.д.) к элементу.

Пример: img { filter: grayscale(100%); }

Документация

Свойство backdrop-filter

Описание: Применяет графические эффекты к фону, находящемуся за элементом.

Пример: div { backdrop-filter: blur(5px); }

Документация

Свойство aspect-ratio

Описание: Определяет соотношение сторон для элемента.

Пример: div { aspect-ratio: 16 / 9; }

Документация

Свойство @keyframes

Описание: Определяет ключевые кадры для CSS-анимации. Используется совместно с animation.

Пример: @keyframes fadeIn {
from { opacity: 0; }
to { opacity: 1; }
}

Документация

Свойство animation

Описание: Позволяет задавать анимацию для элемента с помощью ключевых кадров @keyframes.

Пример: div { animation: fadeIn 1s ease forwards; }

Документация

Свойство animation-delay

Описание: Устанавливает задержку перед началом анимации.

Пример: div { animation-delay: 0.5s; }

Документация

Свойство animation-iteration-count

Описание: Определяет, сколько раз должна проигрываться анимация. Можно указать infinite для бесконечного повтора.

Пример: div { animation-iteration-count: infinite; }

Документация

Свойство will-change

Описание: Подсказывает браузеру, какие свойства элемента могут измениться в будущем, для подготовки оптимизации рендеринга.

Пример: div { will-change: transform; }

Документация

Свойство transition

Описание: Устанавливает плавный переход между двумя состояниями элемента при изменении CSS-свойств.

Пример: button { transition: background-color 0.3s ease; }

Документация

Свойство display: flex

Описание: Включает flex-контейнер, внутри которого элементы автоматически становятся флекс-элементами с гибким позиционированием.

Пример: div { display: flex; }

Документация

Свойство justify-content

Описание: Определяет выравнивание флекс-элементов вдоль главной оси.

Пример: div { justify-content: center; }

Документация

Свойство align-items

Описание: Выравнивает элементы по поперечной оси внутри флекс-контейнера.

Пример: div { align-items: center; }

Документация

Свойство flex-wrap

Описание: Управляет переносом флекс-элементов на новую строку, если они не помещаются.

Пример: div { flex-wrap: wrap; }

Документация

Свойство flex

Описание: Краткая запись для flex-grow, flex-shrink и flex-basis. Управляет тем, как элемент растет, сжимается и занимает пространство.

Пример: div { flex: 1; }

Документация

Свойство display: grid

Описание: Включает Grid Layout для контейнера, создавая двумерную сетку для размещения элементов.

Пример: div { display: grid; }

Документация

Свойство grid-template-columns

Описание: Определяет количество и ширину колонок в grid-сетке.

Пример: div { grid-template-columns: repeat(3, 1fr); }

Документация

Свойство gap

Описание: Задает расстояние между строками и колонками в Grid или Flex контейнере.

Пример: div { gap: 20px; }

Документация

Свойство position

Описание: Определяет метод позиционирования элемента относительно его родителя или документа. Может быть static, relative, absolute, fixed, sticky.

Пример: div { position: absolute; top: 10px; left: 20px; }

Документация

Свойство top/right/bottom/left

Описание: Определяют смещение элемента относительно его контейнера. Используется с позиционированием absolute, relative, fixed.

Пример: div { position: absolute; top: 20px; left: 30px; }

Документация

Свойство z-index

Описание: Определяет порядок наложения элементов, если они пересекаются. Элементы с более высоким значением z-index будут отображаться поверх элементов с более низким значением.

Пример: div { position: absolute; z-index: 10; }

Документация

Свойство background-position

Описание: Определяет начальную позицию фонового изображения в контейнере.

Пример: div { background-position: center; }

Документация

Свойство background-size

Описание: Определяет размер фонового изображения. Можно использовать значения в пикселях, процентах, или специальные ключевые слова, как cover или contain.

Пример: div { background-size: cover; }

Документация

Свойство background-attachment

Описание: Определяет, будет ли фоновое изображение прокручиваться вместе с содержимым страницы или оставаться на месте.

Пример: div { background-attachment: fixed; }

Документация

Свойство object-fit

Описание: Определяет, как содержимое элемента (например, изображения или видео) будет масштабироваться внутри его контейнера.

Пример: img { object-fit: cover; }

Документация

Свойство object-position

Описание: Определяет начальное положение содержимого (например, изображения или видео) в пределах контейнера, когда используется свойство object-fit.

Пример: img { object-position: center; }

Документация

Свойство transform

Описание: Позволяет применять 2D и 3D трансформации к элементам, такие как перемещения, масштабирование, наклон и повороты.

Пример: div { transform: rotate(45deg); }

Документация

Свойство rotate()

Описание: Поворачивает элемент на указанный угол в градусах (например, deg).

Пример: div { transform: rotate(90deg); }

Документация

Свойство scale()

Описание: Изменяет размер элемента, увеличивая или уменьшая его.

Пример: div { transform: scale(1.5); }

Документация

Свойство translate()

Описание: Перемещает элемент по осям X и Y на заданное расстояние.

Пример: div { transform: translate(50px, 100px); }

Документация

Свойство skew()

Описание: Наклоняет элемент по осям X и Y на заданные углы.

Пример: div { transform: skew(30deg, 20deg); }

Документация

Свойство filter

Описание: Применяет графические эффекты, такие как размытие, изменение яркости, контраста и других параметров изображения или элемента.

Пример: img { filter: blur(5px); }

Документация

Функция blur()

Описание: Применяет эффект размытия к элементу.

Пример: img { filter: blur(8px); }

Документация

Функция brightness()

Описание: Регулирует яркость изображения. Значение 100% — это исходное изображение, значение меньше 100% делает изображение темнее, а больше — ярче.

Пример: img { filter: brightness(150%); }

Документация

Функция contrast()

Описание: Управляет контрастностью изображения. Значение 100% — это исходное изображение.

Пример: img { filter: contrast(200%); }

Документация

Функция grayscale()

Описание: Применяет эффект монохромности к изображению, преобразуя его в черно-белое.

Пример: img { filter: grayscale(100%); }

Документация

Функция sepia()

Описание: Применяет эффект сепии к изображению, придавая ему старинный вид.

Пример: img { filter: sepia(100%); }

Документация

Свойство font-family

Описание: Определяет шрифт, который будет использоваться для текста. Можно указать несколько шрифтов через запятую, чтобы браузер выбрал первый доступный.

Пример: p { font-family: Arial, sans-serif; }

Документация

Свойство font-size

Описание: Определяет размер шрифта. Можно использовать различные единицы измерения: пиксели (px), эм (em), проценты (%), рем (rem).

Пример: p { font-size: 16px; }

Документация

Свойство line-height

Описание: Определяет высоту строки, что влияет на расстояние между строками текста в абзаце.

Пример: p { line-height: 1.5; }

Документация

Свойство font-weight

Описание: Определяет жирность текста. Может принимать значения от normal до bold или числовые значения от 100 до 900.

Пример: p { font-weight: bold; }

Документация

Свойство font-style

Описание: Определяет стиль шрифта, например, наклонный (курсив) или обычный.

Пример: p { font-style: italic; }

Документация

Свойство text-transform

Описание: Определяет, как будет отображаться текст: все буквы в верхнем регистре, в нижнем, или первая буква каждого слова с заглавной.

Пример: p { text-transform: uppercase; }

Документация

Свойство letter-spacing

Описание: Определяет расстояние между символами текста.

Пример: p { letter-spacing: 2px; }

Документация

Свойство word-spacing

Описание: Определяет расстояние между словами в тексте.

Пример: p { word-spacing: 5px; }

Документация

Свойство text-align

Описание: Определяет выравнивание текста по горизонтали. Можно выравнивать текст по левому краю, по центру, по правому краю или по ширине.

Пример: p { text-align: center; }

Документация

Свойство text-indent

Описание: Определяет отступ первой строки текста.

Пример: p { text-indent: 20px; }

Документация

Свойство text-decoration

Описание: Добавляет текстовые украшения, такие как подчеркивание, перечеркивание или надчеркивание.

Пример: a { text-decoration: underline; }

Документация

Свойство text-shadow

Описание: Добавляет тень к тексту. Можно задать смещение по осям X и Y, а также цвет и размытие тени.

Пример: p { text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3); }
